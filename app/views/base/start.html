Use this service if you want to register and apply for a licence to become an author. An author is someone who writes books. You can apply to be an author for fiction, non-fiction or scientific books.

You can use this to:

  • register for an account
  • apply for a licence to write one or more style of book
  • check the progress of your application
  • change personal details or the type of book you want to write

Before you start
